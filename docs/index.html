How to start

Connect, upload and see...

Setup your IQRF network If you do not have any existing IQRF network you will need to build one. There is a great IQRF Startup Kit DS-START-04 doing thios job for you. 

Conect IQRF BB-01 development board Connect your development board as an end node. You do not need to connect any thing to board, just plug-in IQRF module and run it. You can follow our Quick Start.

Connect your external sensors Connect sensors and write software. It's great to start with some of sensor or actuator in  ours examples on our Github.  Then you can modify code or change sensor. If you need any help please feel free to call us.

Is your prototype perfectly working? It's time to think about  product. Just use development board drawing as a good strat. If you are not good at hardware design we are here to help you. We can also ensure certificaton of your end device so any gateway in the world would be aware of your device. Contact Logimic.
